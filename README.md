## Usage

### Andy's Blessing

Create a blessing for Andy:

```bash
python scripts/bless_andy.py
```

Or use as a module:

```python
from apollo.blessings import ApolloAndyBlessing

blessing = ApolloAndyBlessing()
blessing.create_andy_blessing()
```
