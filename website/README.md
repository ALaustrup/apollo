# The Apollo Foundation for Creative Kindness - Website

Spread kindness. Warm the cold world. Bring joy through giving.

## Technology

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS (via inline styles and CSS modules)
- **Hosting**: Vercel

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Deployment

This website is designed to be deployed on Vercel. Simply connect your GitHub repository to Vercel and it will automatically deploy.

## Features

- Modern, beautiful UI with gradient effects
- Responsive design
- Smooth animations
- Mission and values presentation
- Activities showcase
- Integration ready for Demiurge Blockchain
